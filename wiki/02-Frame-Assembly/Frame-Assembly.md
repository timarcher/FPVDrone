# Build Notes
- Make sure you get 12mm motors so that they will mount properly to the frame.
- You can 3d print taller landing arm mounts: [iFlight Protek 35 - Landing-Train / Landing-Train](https://cults3d.com/en/3d-model/various/iflight-protek-35-landing-train-aterrizaje-tren). Before printing, I had to scale the X and Y axes to 75% in my Creality slicer tool so that it would mount to the frame. Leave the Z axis at 100%.
- Top Plate Riser you can 3D print if you want to get some more clearance inside the drone frame: [iFlight Protek35 Top Plate Raiser](https://www.thingiverse.com/thing:6373708)
- [Protek 35 DJI O3 Air Unit Camera Mount v2](https://cults3d.com/en/3d-model/tool/protek-35-dji-o3-air-unit-camera-mount-v2)
